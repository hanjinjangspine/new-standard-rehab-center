# New Standard Hospital Design Alignment V1

## 1. 통합 브랜드 원칙

새기준병원 계열 사이트는 동일한 병원 브랜드처럼 보여야 하지만, 각 사이트의 역할과 환자군에 따라 톤은 다르게 유지한다.

공통 키워드:

- 신뢰감
- 정돈됨
- 전문성
- 환자 친화성
- 검사와 진찰 기반 상담
- 과장 없는 설명

모든 환자용 화면은 치료를 과장하거나 특정 치료를 단정적으로 권유하기보다, 증상과 진찰 소견, 영상검사 결과를 함께 확인한 뒤 치료 방향을 상담한다는 흐름을 유지한다.

## 2. 공통 컬러 토큰

| Token | Hex | 용도 |
| --- | --- | --- |
| Deep Navy | `#10283D` | 헤더, 푸터, 진중한 CTA, 전문성 강조 |
| Hospital Green | `#2B7366` | 병원 브랜드 중심색, primary CTA |
| Mint Accent | `#3ABFB0` | 포인트, hover, 보조 강조 |
| Soft Gray | `#F8FAFB` | 섹션 배경, 차분한 면 구성 |
| Border Gray | `#D9E2E7` | 카드, 구분선, 입력 영역 |
| Academic Gold | `#9A7B38` | 학술사이트 포인트, 교육/학술 강조 |

## 3. 사이트별 컬러 방향

### 본원

New Standard Green + White + Deep Navy

공식 병원 포털형 톤을 유지한다. 가장 공신력 있고 넓은 진료 범위를 담는 사이트이므로, 과도한 장식보다 명확한 정보 구조와 안정적인 색 대비를 우선한다.

### 관절센터

Deep Navy + Green Accent + Light Mint + White

족부·발목, 무릎, 어깨 전문센터형 톤을 유지한다. 카드형 안내와 진료 영역 구분이 선명해야 하며, 본원과 연결된 독립 센터처럼 보여야 한다.

### 회복재활센터

Hospital Green + Warm Mint + Cream/Ivory + Soft Gray

생활 통증과 수술 후 회복관리형 톤을 유지한다. 실제 치료실 사진, 부드러운 색감, 환자 친화적인 문장을 활용하되, 도수치료 판매형 인상은 피한다.

### 학술사이트

Deep Navy + Academic Gold + White

논문·증례·학술 브랜딩형 톤을 유지한다. 환자용 사이트와 다르게 academic tone을 허용하되, 본원과의 연결성은 footer와 관련 리소스 섹션에서 유지한다.

### BBS 건강칼럼

Green + White + Soft Gray

본원 공식 의료 콘텐츠 허브형 톤을 유지한다. 단순 게시판이 아니라 새기준병원 의료진이 정리한 공식 건강칼럼처럼 보여야 한다.

### 관절칼럼

Deep Navy + Green Accent + White

관절센터 공식 콘텐츠형 톤을 유지한다. 관절 질환, 통증, 회복관리 정보를 환자 관점에서 쉽게 설명하는 콘텐츠 허브로 정리한다.

## 4. 공통 UI 기준

### 로고

환자용 사이트는 본원 공식 로고를 동일 비율로 사용한다.

- 본원 로고 경로: `/assets/img/common/logo.png`
- 관절센터/회복재활센터 로고 경로: `/brand/newstandard-logo-horizontal.png`

학술사이트는 `Hanjin Jang, MD` 텍스트 로고를 유지할 수 있다. 다만 footer 또는 기관 소개 영역에서 New Standard Hospital과의 연결을 명확히 유지한다.

### 버튼

- Primary CTA는 green/navy 계열을 사용한다.
- Radius는 `rounded-md`부터 `rounded-2xl` 범위에서 사용한다.
- 과도한 gradient는 사용하지 않는다.
- CTA 문구는 "상담", "안내", "오시는 길" 중심으로 정리한다.
- 환자에게 즉각적인 치료 결과를 기대하게 하는 문구는 피한다.

권장 CTA 예시:

- 진료 상담
- 센터 안내
- 오시는 길
- 치료 방향 상담
- 회복관리 안내

### 카드

- Radius는 12~16px 범위를 기본으로 한다.
- Border는 `#D9E2E7` 또는 유사 light border를 사용한다.
- Shadow는 약하게 사용한다.
- 배경은 white 또는 soft gray를 기본으로 한다.
- 모바일에서는 1열 안정성을 우선한다.
- 카드 안의 텍스트는 과도하게 작게 만들지 않는다.

### Typography

- H1은 한 페이지 1개만 사용한다.
- 모바일에서 긴 H1은 줄바꿈을 제어한다.
- 본문은 환자 이해도를 우선한다.
- 환자용 사이트는 Korean readability를 우선하고, 과도한 letter spacing을 피한다.
- 학술사이트만 serif/acadmic tone을 허용한다.

### Footer

Footer에는 다음 정보가 일관되게 포함되어야 한다.

- 새기준병원 주소
- 대표전화 `031-328-0333`
- 본원, 관절센터, 회복재활센터 연결
- 공식채널 링크
- 의료 정보는 진료와 검사 결과에 따라 달라질 수 있다는 안전 문구

## 5. 사이트별 차별화 기준

### 본원

가장 공식적이고 신뢰감 있는 병원 포털이다. 척추센터, 관절센터, 회복재활센터, 건강칼럼, 의료진, 오시는 길로 이어지는 중심 허브 역할을 한다.

### 관절센터

관절·족부·발목 진료 특화 사이트다. 무릎, 어깨, 손·손목, 족부·발목 통증을 카드형 구조로 선명하게 안내하고, 본원과 회복재활센터로 자연스럽게 연결한다.

### 회복재활센터

따뜻하고 부드러운 회복형 사이트다. 생활 통증, 급성 염좌, 산후·육아 통증, 직장인 통증, 고령자 보행, 척추·관절 수술 후 회복관리를 실제 치료실 사진과 함께 안내한다.

### 학술사이트

장한진 대표원장의 학술성과와 비식별 증례 기반 교육자료 중심 사이트다. 환자용 상담 페이지가 아니라 academic and educational resource로 유지한다.

## 6. 안전 문구 기준

공통 권장 문구:

- "증상, 진찰 소견, 영상검사 결과를 함께 확인해 치료 방향을 상담합니다."
- "환자 상태에 따라 치료 방법은 달라질 수 있습니다."
- "모든 환자에게 같은 치료가 적용되지는 않습니다."
- "치료 방향은 진료와 검사 결과를 종합해 상담합니다."
- "검사 결과와 기존 치료 반응을 함께 확인합니다."

금지 표현:

- 완치
- 100%
- 최고
- 즉시 회복
- 부작용 없음
- 보장
- 유일
- 무조건
- 반드시 낫
- 완벽

## 7. 수정 우선순위

1. 회복재활센터 Project 3에 디자인 기준 우선 반영
2. 관절센터 hero/CTA/이미지 신뢰 요소 미세 조정
3. 본원 BBS/건강칼럼 카드와 CTA를 최신 `nsh*` 톤으로 정리
4. 본원 공통 헤더/푸터는 장기 전면개편 때 반영
5. 학술사이트는 tone은 유지하되 footer와 본원 연결만 브랜드 정합성 유지

## 8. 향후 작업 원칙

- 한 번에 한 사이트만 수정한다.
- 한 번에 1~2개 파일만 수정한다.
- 공통 파일은 원인 확인 전 수정하지 않는다.
- 수정 전 백업 또는 git clean 상태를 확인한다.
- 수정 후 build를 실행한다.
- 실제 화면과 소스 기준으로 검수한다.
- noindex/index 설정은 별도 단계에서만 변경한다.
- 환자용 사이트와 학술사이트의 역할을 섞지 않는다.
- 외부 링크는 필요한 경우 `target="_blank"`와 `rel="noopener noreferrer"`를 적용한다.
